Zomato Restaurant Analysis
Project Overview

This project analyzes restaurant data collected through web scraping to uncover customer preferences, restaurant quality patterns, cuisine trends, and hidden growth opportunities.

Objectives
Analyze restaurant ratings and customer reviews
Identify popular cuisine categories
Discover hidden gem restaurants
Segment restaurants by quality and popularity
Perform hypothesis testing to validate insights
Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
SciPy
Jupyter Notebook
Key Insights
North Indian cuisine dominates restaurant offerings.
Most restaurants fall within Good and Very Good rating categories.
Hidden Gem restaurants achieve higher ratings than the platform average.
Customer engagement is concentrated among highly popular restaurants.
Hypothesis Testing
One-Sample T-Test
ANOVA
Chi-Square Test
Business Recommendations
Promote Hidden Gem restaurants.
Improve cuisine-based recommendations.
Increase visibility for high-quality low-review restaurants.
